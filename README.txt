Wordpress plugin to post messages for new blog posts in discord channels.

== Description ==

Wordpress Plugin used to post messages across discord channels using a bot.

== Installation ==

1. Upload the `wp-discord` folder to your `/wp-content/plugins/` directory
2. Activate the plugin using the `Plugins` menu in WordPress
3. Go to WP Discord (Toolbar)
4. Adjust Options as You Prefer
5. Create an application at [Discord](https://discordapp.com/developers/applications/me/create) and Configure your OAuth keys
